Local Weather

Overview

Local Weather is a web app that displays current weather data as well as weather data for a 5-day forecast and shows a background image that matches the current weather conditions in their location. The user can toggle the temperature displayed between Farenheit and Celsius as well as the wind speed between mph and kph.

Purpose and Context

The Local Weather App is a personal project I created as a part of my Legacy Front End Certification course on Free Code Camp to showcase my JavaScript skills and ability to incorporate APIs.

Objective

The goal of this project was to build a web app using a library or API to show the weather in the user's location, displays a different icon or background image based on the weather, and lets the user toggle between Farenheit and Celsius.

Details

Technologies: HTML, CSS, Bootstrap, JavaScript, jQuery, National Weather Service API

Duration: 1 month
